오늘 배운 내용

Branch 

    branch란

Git은 branch 명령어로 각기 다른 공간에서 서로 작업이 가능함
같은 이름의 A 파일이라도 다른 공간이므로 영향이 끼치지 않음
이 후 merge 명령어를 통해 두 공간을 병합(merge) 가능

    명령어 리스트

  • git branch : 현재 존재하는 branch 공간을 출력한다.

  • git branch -D ‘name’ : ‘name’의 branch를 삭제한다.

  • git merge ‘name’ : ‘name’의 branch를 서로 병합한다.

  • git switch ‘name’ : ‘name’의 branch로 이동한다. (구버전은 checkout)

      주의사항

branch로 작업후 꼭 add, commit을 하도록 하자.
commit을 하지 않으면 두 공간에서 같은 파일이 출력된다.
